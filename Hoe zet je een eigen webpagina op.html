Hoe zet je een webpagina op

Je begint de pagina met een programma zoals notepad++. Je slaat het bestand op en geeft het een naam zonder spaties met hierachter geschreven '.HTML'. Dan kopieer en plak je het volgende:

<!DOCTYPE html>
<html>
<head>
<title>pagina titel</title>
</head>
<body>
<h1>dit is een kop.</h1>
<p>dit is een paragraaf.</p>
</body>
</html>

Naast een HTML pagina kan je ook nog een CSS pagina maken. Deze wordt gebruikt voor de opmaak van de pagina en alles wat te maken heeft met het ontwerpen ervan. Het kan toegepast worden op meerdere pagina's.

Het commando 'text-align' wordt gebruikt om de plaats van de tekst te bepalen.

'font-family' wordt gebruikt om aan te geven in welk lettertype de tekst wordt vermeld.

Het commando 'font-style' wordt enkel gebruikt om tekst schuingedrukt te maken.

Met het commando 'font-size' kan een geselecteerd stuk tekst groter of kleiner gemaakt worden.

Het 'em' commando doet eigenlijk hetzelfde als 'font-style', alhoewel 'em' voornamelijk wordt toegepast op kleinere stukken tekst en 'font-style' op grotere stukken tekst.

Met het 'background' command kan de achtergrond aangepast worden. Dit kan bijvoorbeeld door het te vervangen met een plaatje of een kleur.

Het 'color' command zorgt ervoor dat je de kleur van de letters kan veranderen.

Commentaar in het HTML bestand zelf kan vermeld worden met <!--, en dit sluit je weer af met -->.
